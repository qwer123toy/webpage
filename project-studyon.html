StudyOn

효과적인 학습 루틴 관리와 목표 달성을 지원하는 Android 애플리케이션

프로젝트 개요

StudyOn은 개인의 학습 목표를 설정하고, 일일 루틴을 관리하며, 학습 통계를 시각화하여 지속적인 학습 동기를 제공하는 안드로이드 앱입니다. 로컬 데이터베이스를 활용하여 개인정보 보호와 빠른 접근성을 보장하며, MVVM 아키텍처와 최신 Android 개발 기술을 적용했습니다.

주요 기능

  • 홈 화면: 오늘의 루틴 확인 및 체크, 일일 학습 시간 확인
  • 목표 관리: 학습 목표 설정, 할 일 목록 생성 및 추적, 마감일 및 태그 기반 분류
  • 루틴 기록: 학습 시간 타이머, 수동 시간 입력, 학습 내용 메모
  • 통계 분석: 주간/월간 학습 패턴, 목표별 시간 분포 차트, 시간대별 집중도 히트맵
  • 성취 시스템: 학습 배지 및 어워드, 연속 학습 달력, 목표 달성률 추적

기술 스택

아키텍처

  • MVVM (Model-View-ViewModel) 패턴
  • Repository 패턴을 통한 데이터 계층 분리

사용 기술

  • Room Database: 로컬 데이터 저장
  • Navigation Component: 프래그먼트 네비게이션
  • LiveData & ViewModel: 반응형 UI 및 생명주기 관리
  • View Binding & Data Binding: 안전한 뷰 접근
  • MPAndroidChart: 통계 차트 시각화
