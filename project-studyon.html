StudyOn

효과적인 학습 루틴 관리와 목표 달성을 지원하는 Android 애플리케이션

🎯 프로젝트 목표

학습자들이 자신의 학습 목표를 효과적으로 설정하고 관리하며, 일일 루틴을 통해 꾸준한 학습을 이어갈 수 있도록 돕는 것이 이 프로젝트의 주요 목표입니다. 특히 학습 통계를 시각화하여 사용자가 자신의 학습 패턴을 파악하고 개선할 수 있도록 지원합니다.

💡 주요 기능

홈 화면

  • 오늘의 학습 루틴 확인 및 체크 기능
  • 일일 학습 시간 실시간 트래킹
  • 빠른 루틴 시작을 위한 직관적인 인터페이스
  • 진행 중인 목표 달성률 시각화

목표 관리

  • 상세한 학습 목표 설정 (제목, 설명, 기간, 우선순위)
  • 할 일 목록 생성 및 진행 상태 추적
  • 마감일 기반의 자동 알림 시스템
  • 태그 시스템을 통한 효율적인 목표 분류

루틴 기록

  • 정밀한 학습 시간 측정 타이머
  • 수동 시간 입력 옵션 제공
  • 학습 내용 메모 및 태그 기능
  • 목표별 학습 시간 자동 분류

통계 분석

  • 주간/월간 학습 패턴 분석 그래프
  • 목표별 투자 시간 분포도
  • 시간대별 학습 집중도 히트맵
  • 학습 시간 누적 통계 제공

🛠️ 개발 환경 및 기술 스택

개발 환경

  • Android Studio Arctic Fox 이상
  • Kotlin 1.5.0+
  • Minimum SDK: API 24 (Android 7.0)
  • Target SDK: API 34 (Android 14)

사용 기술

  • MVVM 아키텍처 패턴
  • Android Jetpack Components
    • Room Database - 로컬 데이터 저장
    • Navigation Component - 화면 전환
    • LiveData & ViewModel - 데이터 바인딩
    • WorkManager - 백그라운드 작업
  • Coroutines - 비동기 처리
  • MPAndroidChart - 통계 차트 구현
  • Hilt - 의존성 주입

📱 사용자 인터페이스

  • Material Design 3 가이드라인 준수
  • 다크 모드 지원
  • 동적 테마 시스템 적용
  • 직관적인 제스처 기반 인터랙션
  • 접근성 가이드라인 준수

🔒 데이터 보안

  • 로컬 데이터베이스 암호화
  • 민감 정보 보호를 위한 EncryptedSharedPreferences 사용
  • 백업 및 복원 기능 제공

🌟 프로젝트 특징

  • Clean Architecture 원칙 준수
  • 단위 테스트 및 UI 테스트 구현
  • 지속적인 성능 최적화
  • 사용자 피드백 기반 기능 개선

  • 안정적인 사용자 경험 제공
