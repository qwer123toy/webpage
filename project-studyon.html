StudyOn - 학습 루틴 관리 앱

효과적인 학습 루틴 관리와 목표 달성을 지원하는 Android 애플리케이션

프로젝트 개요

StudyOn은 개인의 학습 목표를 설정하고, 일일 루틴을 관리하며, 학습 통계를 시각화하여 지속적인 학습 동기를 제공하는 안드로이드 앱입니다.
로컬 데이터베이스를 활용하여 개인정보 보호와 빠른 접근성을 보장하며, MVVM 아키텍처와 최신 Android 개발 기술을 적용했습니다.

주요 기능

  • 홈 화면 : 오늘의 루틴 확인/체크, 일일 학습 시간 실시간 트래킹
  • 목표 관리 : 상세 학습 목표 설정, 할 일 목록 생성 및 진행 상태 추적
  • 루틴 기록 : 정밀한 학습 시간 측정, 메모 및 태그 기능
  • 통계 분석 : 주간/월간 학습 패턴, 목표별 시간 분포 차트
  • 성취 시스템 : 학습 배지, 연속 학습 달력, 목표 달성률 추적
  • 설정 : 다크 모드, 알림 설정, 데이터 백업/복원

기술적 특징

  • MVVM 아키텍처 : 깔끔한 코드 구조와 유지보수성 향상
  • Room Database : 효율적인 로컬 데이터 관리
  • Android Jetpack : 최신 Android 개발 도구 활용
  • Material Design 3 : 모던하고 직관적인 UI/UX
  • 데이터 시각화 : MPAndroidChart를 활용한 통계 차트

사용 기술

Kotlin Android MVVM Room DB Jetpack
  • 개발 환경 : Android Studio Arctic Fox, Kotlin 1.5.0+
  • 아키텍처 : MVVM, Repository 패턴, Clean Architecture
  • 데이터베이스 : Room DB, EncryptedSharedPreferences
  • 비동기 처리 : Coroutines, WorkManager

구현 과정

  • 사용자 요구사항 분석 및 UI/UX 설계
  • MVVM 아키텍처 기반 프로젝트 구조화
  • Room Database 스키마 설계 및 구현
  • Android Jetpack 컴포넌트 통합
  • Material Design 3 가이드라인 적용
  • 성능 최적화 및 메모리 관리

프로젝트 성과

  • Google Play 스토어 출시 및 안정적인 운영
  • Clean Architecture를 통한 확장 가능한 코드베이스 구축
  • 효율적인 학습 관리 도구 제공으로 사용자 만족도 향상
  • 지속적인 피드백 반영 및 기능 개선
