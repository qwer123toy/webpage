법인카드·차량 이용 관리 시스템

사내 카드 지출과 차량 사용 신청을 통합하여 효율적으로 관리할 수 있는 사이트

프로젝트 개요

본 시스템은 기업 내 법인카드 사용 결의서 작성과 개인 차량 이용 신청 과정을 전산화하여, 문서 관리, 결재, 출력, 통계 기능을 하나로 통합한 웹 사이트입니다.
기존의 수기 작성 및 개별 파일 관리 방식에서 벗어나 중복 업무를 줄이고 결재 프로세스를 간소화하여 관리 효율성을 극대화하였습니다. 카드 지출과 차량 이용 데이터를 통합 관리함으로써 부서별, 목적별 통계 집계 및 투명한 결재 흐름이 가능해졌습니다.

주요 기능

  • 사용자 기능 : 로그인 및 회원가입, 카드 사용 내역 등록/조회/수정, 차량 이용 신청/수정/출력
  • 관리자 기능 : 사용자 계정 활성/비활성 관리, 카드/차량 이용 내역 검토 및 통계 출력, 유류비 단가 설정
  • 출력 및 결재 : 카드 지출 결의서 PDF 출력, 차량 신청서 출력, 다단계 결재선 지정 및 상태 시각화

사용 기술

ASP (Classic) MSSQL JavaScript HTML CSS Bootstrap ShadCN 스타일
  • 백엔드 : Classic ASP 기반 서버 사이드 스크립트
  • DB : MSSQL Server를 RDBMS로 사용, 테이블 기반 데이터 저장 및 조회 처리
  • 프론트엔드 : JavaScript, HTML, CSS, Bootstrap
  • 스타일 시스템 : 일부 화면에 ShadCN 스타일을 도입해 일관된 관리자 UI 제공
  • 서버 환경 : Windows 서버 + IIS 환경에서 구동

개발 절차 및 방법론

  • 요구사항 수집 → 업무 흐름 분석 → 프로세스 도식화
  • ERD 설계 및 테이블 정의서 기반 MSSQL 테이블 구성
  • 모듈 단위로 ASP 페이지 설계 및 개발
  • 다단계 결재는 approvallogs 테이블 기반으로 설계
  • 파일 업로드, 인쇄, 금액 자동 계산, 결재 상태 표시 등 실무 기능 포함

내 역할

  • 전체 ASP 페이지 설계 및 개발
  • 카드 사용 등록/수정/삭제, 차량 신청서 입력 및 출력 기능 일체 구현
  • Bootstrap + ShadCN 스타일 기반 UI 커스터마이징
  • 세션 기반 로그인 및 승인 권한 처리 로직 작성
  • 다단계 결재선 지정 팝업 UI + DB 연동 로직 개발
