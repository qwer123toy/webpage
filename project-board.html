Spring Boot 익명 게시판

익명 기반으로 글을 작성하고 비밀번호로 보호할 수 있는 Q&A 게시판

프로젝트 개요

본 프로젝트는 사용자 로그인을 요구하지 않고 누구나 자유롭게 글을 작성할 수 있는 익명 게시판으로, 비밀번호 기반 접근 제어를 통해 수정/삭제 권한을 제한하고, 사용자 간의 정보 공유와 소통이 가능하도록 구성된 Q&A 시스템입니다.

주요 기능

  • 게시글 기능 : 글 작성, 조회, 수정, 삭제
  • 보안 처리 : 비밀글 설정, 비밀번호 인증 기반 수정/삭제
  • 검색 및 페이징 : 키워드 기반 검색 및 페이지 이동
  • 예외 처리 : 404 에러 핸들링, 사용자 피드백 출력

사용 기술

Java Spring Boot JPA JSP H2 DB JSTL DBCP2 JUnit
  • 백엔드 : Spring Boot, JPA 기반 게시판 로직 구현
  • DB : H2 인메모리 데이터베이스 활용
  • 프론트엔드 : JSP, JSTL, HTML, CSS
  • 테스트 : JUnit을 이용한 기능 테스트

개발 절차 및 방법론

  • 요구사항 도출 및 익명 게시판 흐름 설계
  • Controller - Service - Repository 구조 설계
  • 기능 단위 구현 후 JSP 기반 화면 연결
  • 비밀번호 인증 처리 및 오류 대응 로직 구현
  • JUnit 기반 단위 테스트를 통한 검증

내 역할

  • 익명 Q&A 게시판 전체 아키텍처 설계
  • CRUD 기능 구현 및 비밀번호 인증 로직 처리
  • 검색/페이징 기능과 예외 처리 구성
  • 테스트 코드 작성 및 화면 검증
